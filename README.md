# Personal Quote Service

## Description

Repository for my backend quote REST service written using [Nest](https://github.com/nestjs/nest) framework and mongodb.

## Installation

Docker install (https://docs.docker.com/install/)

Using docker to run local instance of mongodb.

```bash
$ npm install
```

## Running the app

```bash
# running docker mongodb
$ docker-compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
